# GoalFlow Starter

Simple Vue 3 + TypeScript starter project for the GoalFlow homework.

## What is included

- Goals page
- Projects page
- Tasks page
- Simple timer start/stop
- Task filtering by status
- Local storage in browser
- JSON export
- Pinia stores
- Vue Router

## Tech stack

- Vue 3
- TypeScript
- Vite
- Pinia
- Vue Router
- localStorage

## Install

```bash
npm install
npm run dev
```

## Simple app flow

1. Create a goal
2. Create a project inside the goal
3. Create a task inside the project
4. Start timer for the task
5. Stop timer
6. Check analytics page
7. Export backup

## Notes

This starter uses `localStorage` to keep the project simple.
You can later replace it with Dexie + IndexedDB.
